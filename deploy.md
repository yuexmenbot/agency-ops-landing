# Deployment Instructions

## GitHub Pages Setup

The repository has been created at: https://github.com/yuexmenbot/agency-ops-landing

To complete the GitHub Pages deployment:

1. Go to https://github.com/yuexmenbot/agency-ops-landing/settings/pages
2. Under "Source", select "Deploy from a branch"
3. Select "main" branch and "/ (root)" folder
4. Click "Save"

The site will then be available at: https://yuexmenbot.github.io/agency-ops-landing
